# OptiHire 2.0 - AI Job Matching Chrome Extension

A powerful Chrome extension that analyzes how well your LinkedIn profile matches with job postings using AI-powered skills matching.

## Features

- **One-Click Profile Sync**: Extract and store your LinkedIn profile data securely
- **Real-Time Job Analysis**: Get instant match scores on LinkedIn job postings
- **Skill Matching**: See matched skills, missing skills, and personalized recommendations
- **Match History**: Track all analyzed jobs and your match scores
- **Beautiful UI**: Clean, professional sidebar that doesn't interfere with LinkedIn
- **Privacy-First**: All data stored locally in Chrome storage
